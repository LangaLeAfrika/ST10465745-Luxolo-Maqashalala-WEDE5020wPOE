# Eyethu Website

Eyethu is a retail business centre based in Joe Slovo, Port Elizabeth, South Africa. The development consists of **Eyethu Filling Station**, **Eyethu Superstore**, and **Eyethu Shopping Centre** (8 retail shops).

This website is designed to provide a digital presence for Eyethu, showcasing services, promotions, and community engagement.

## üåü Features and Functionality (Project Proposal Aspect)
This website provides the following features and core functionality, designed to meet user needs:

* **Usability:** Clear navigation menu for easy access to all pages (Home, About Us, Our Services, Enquiry, Contact Us).
* **Information Access:** Dedicated pages detailing the business profile, services offered (Filling Station, Superstore, Retail Shops), and contact information.
* **Interactivity:** An Enquiry Form for collecting user questions and a dedicated JavaScript file (`script.js`) for handling interactive elements.
* **Location Awareness:** An embedded Google Maps placeholder (`<iframe>`) on the Contact Us page to help users find the physical location.
* **Responsiveness:** The CSS (`style.css`) is structured to ensure the site is viewable across different devices.
